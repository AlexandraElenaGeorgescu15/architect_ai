‚ùì Common Questions (Code-Based Answers)

Q: How does Knowledge Graph work?
A: Uses ast.parse() for Python and regex for TS/C#/Java to extract classes, functions, imports, and dependencies. Builds a NetworkX directed graph. Calculates coupling (density), clustering coefficient, and complexity scores. Outputs YOUR actual component relationships.
Q: What patterns can Pattern Mining detect?
A: Design Patterns: Singleton, Factory, Observer, Strategy, Repository
Anti-Patterns: God Class (>500 LOC), Long Method (>50 LOC), Duplicate Code
Code Smells: Magic Numbers, Dead Code, Complex Conditionals (>5 branches)
Provides quality score (0-100) and specific refactoring suggestions.
Q: How does multi-agent work in the code?
A: When enabled (use_multi_agent = True), the system creates a MultiAgentOrchestrator that runs 3 specialized agents (Design, Backend, Frontend) from agents/specialized_agents.py. Each agent reviews the output and provides feedback. Final output incorporates all 3 perspectives. Cost: 3x API calls.
Q: Which Ollama models are actually supported?
A: From ai/artifact_router.py:
‚Ä¢ codellama:7b-instruct-q4_K_M (3.8GB VRAM) - Code generation
‚Ä¢ llama3:8b-instruct-q4_K_M (4.7GB VRAM) - Docs, API, JIRA
‚Ä¢ mistral:7b-instruct-q4_K_M (4.1GB VRAM) - Mermaid diagrams
‚Ä¢ qwen2.5-coder:7b-instruct-q4_K_M (4.3GB VRAM) - HTML/CSS prototypes
Q: How does artifact routing actually work?
A: ArtifactModelRouter maintains a map: ArtifactType ‚Üí List[Models] ordered by priority. Example:
ERD ‚Üí [mistral:7b, llama3:8b]
CODE ‚Üí [codellama:7b, qwen2.5-coder:7b]
System tries first model, validates output. If score < 60, tries next model. If all local fail, falls back to cloud (Gemini 2.0 Flash).
Q: Is fine-tuning fully implemented?
A: Yes! components/local_finetuning.py (2433 lines) includes:
‚Ä¢ Feedback collection system (finetuning_feedback.py)
‚Ä¢ Dataset builder with 5000+ examples (finetuning_dataset_builder.py, 5412 lines)
‚Ä¢ LoRA/QLoRA training support
‚Ä¢ Model versioning and rollback
Complete UI in üéì Fine-Tuning tab. Feedback shows count and "Ready to train" status.
