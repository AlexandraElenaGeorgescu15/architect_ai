INFRASTRUCTURE & MLOPS

3.1 Model Dispatch & Local AI

Intelligent Dispatching

Smart Routing: Automatically routes tasks to the best model (e.g., specialized coders for Mermaid).

  • Security Middleware

    Active IP-banning and rate-limiting (403 protection) to prevent API abuse.

  • WebSocket Streaming

    Reliable real-time events for long-running generation tasks, ensuring no timeouts.

  • ChromaDB Vector Store

    Secure, local storage for all project embeddings. Data stays on your machine.

FAQ / The Playbook

Q: "What makes this different from Copilot?"

A: Copilot is a text-completer. Architect.AI is a Symbolic Architect. It builds a graph of your project and clusters files by complexity.

Q: "Is my data safe?"

A: Yes. With Local-First mode, all index data and model execution stay on your machine via ChromaDB and Ollama.

Q: "How do you handle broken code?"

A: We use an Agentic Repair Loop. The validation service automatically fixes syntax errors before you see them.
