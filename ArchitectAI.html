SECTION 2

THE ARCHITECTURE

Symbolic vs Generative

2.1 The Double Brain

Symbolic Brain (Facts)

The Knowledge Graph, Pattern Mining, and RAG. This is static, deterministic data.

It doesn't hallucinate. It knows that User.py imports Auth.py.

Generative Brain (Intuition)

The LLM (Cerebras, Groq, or local Ollama). It takes the facts from the Symbolic brain and turns them into human-like artifacts.

SECTION 3

TECHNICAL DEEP DIVES

The Secret Sauce

3.1 ML Pipeline

  • 1. Metric Extraction: LOC, Import Count, Cyclomatic Complexity.
  • 2. Normalization: StandardScaler ensures large files don't drown small critical ones.
  • 3. K-Means Clustering: Groups files into 4 core archetypes:
    • • Cluster 0: Utilities (Helpers)
    • • Cluster 1: DTOs (Data Models)
    • • Cluster 2: Business Logic (Core)
    • • Cluster 3: Entrypoints (System)
  • 4. Prompt Injection: "Agent, you are looking at a Cluster 2 file."

3.2 The Agentic Flow (ReAct)

1. Input

User asks a complex question.

2. Reasoning

"I need to read main.py to answer this."

3. Tool Use

Calls read_file, search_code, or get_project_overview.

4. Answer

Synthesizes tool output into final response.

3.3 Lifecycle of a User Request

Hybrid RAG & KG

RAG (Search): Uses Vector + BM25 (Keyword) search to find exact code snippets with 99% accuracy.

KG (Filter): Extracts entities to find relevant "islands" in the graph (Map Zoom).

PM & ML

PM (Trigger): Meeting notes trigger code-smell checks (Alarm).

ML (Context): Tells AI if it's viewing a "Complex Engine" or "Simple Helper" (Expert Glasses).

SECTION 5

PRODUCTION & SECURITY

Deployment Architecture

5.1 Enterprise Readiness

Deployment

  • Containerization: Docker-ready with async startup pipeline.
  • Local-First: Prioritizes local Ollama execution for privacy.
  • Async Boot: Parallel bootup of RAG/KG/ML services.

Security Layers

  • IP Banning: StructuredLoggingMiddleware + Rate Limiter.
  • CORS: Strict origin validation middleware.
  • Secrets: Pydantic Settings singleton for ENV management.

5.2 System FAQ

Cloud Outage?

"Fallback Pipeline"

If Groq 500s, we route to local Ollama. User sees nothing.

Scale Limits?

"Importance Maps"

We prioritize high-centrality files, not everything.

Invalid Output?

"Agentic Repair"

Validation Service fixes syntax before rendering.
